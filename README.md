Odaklanarak Oku

Metinleri göz yormayan karanlık bir temada, dikkat dağıtmadan okumak için tasarlanmış basit bir web aracı. Hızlıca metin yapıştırın, rahatça okuyun ve anında istatistiklerini görün.

Amaç

Bu araç, uzun metinleri (makaleler, blog yazıları, e-postalar) karmaşık arayüzler veya dikkat dağıtıcı reklamlar olmadan, sadece içeriğe odaklanarak okuma ihtiyacından doğdu. Göz yorgunluğunu azaltmak için karanlık tema ve ayarlanabilir metin stilleri sunar.

Özellikler

Rahat Okuma Arayüzü: Göz yormayan koyu gri (dark mode) tema.

Otomatik Yapıştırma: Sayfa açıldığında panodaki metni otomatik olarak yapıştırmayı dener (Tarayıcı izni gereklidir).

Metin Kontrolleri:

Yazı Boyutu (Büyüt/Küçült)

Satır Aralığı (Artır/Azalt)

Harf Aralığı (Artır/Azalt)

Anlık Metin İstatistikleri:

Kelime Sayısı

Karakter Sayısı

Cümle Sayısı

Paragraf Sayısı

Hece Sayısı (Yaklaşık)

Satır Sayısı

Sayfa Sayısı (Tahmini)

Okuma Süresi (Tahmini)

Konuşma Süresi (Tahmini)

Tek Tuşla Temizle: Metin alanını hızla boşaltır.

Kullanım

Okumak istediğiniz metni kopyalayın.

odaklanarak-oku.html dosyasını tarayıcınızda açın.

Araç, panonuzdaki metni otomatik olarak yapıştırmayı deneyecektir.

Eğer tarayıcı izin vermezse veya panonuz boşsa, metni textarea alanına manuel olarak yapıştırabilirsiniz (Ctrl+V / Cmd+V).

Üstteki kontrol panelinden metni istediğiniz gibi ayarlayın.

Metin alanının altındaki panelden istatistikleri inceleyin.

Kullanılan Teknolojiler

HTML: Sayfanın temel yapısı.

Tailwind CSS: Hızlı ve modern tasarım için CSS framework'ü.

JavaScript (Vanilla JS): Tüm interaktif fonksiyonlar ve istatistik hesaplamaları.
