# odaklanarak-oku
Metinleri göz yormayan karanlık temada, dikkat dağıtmadan okumak için basit bir web aracı. Font boyutu, satır/harf aralığı gibi ayarları kolayca yapın. Yapıştırdığınız metnin kelime, karakter, cümle, paragraf sayısı, tahmini okuma/konuşma süresi ve sayfa sayısı gibi detaylı istatistiklerini anında görün.
